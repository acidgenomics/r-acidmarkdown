# AcidMarkdown

Toolkit for extending the functionality of [R Markdown][].

## Installation

This is an [R][] package.

```r
install.packages(
    pkgs = "AcidMarkdown",
    repos = c(
        "https://r.acidgenomics.com",
        getOption("repos")
    )
)
```

[r markdown]: https://rmarkdown.rstudio.com/
[r]: https://www.r-project.org/
